
# ( সহজ সরল সিম্পল ) ASSIGNMENT-009

This project is about design an app website for skill share with React from Programming Hero.

## Features
- Email signup
- Google signup
- Skill Course apply form with in private route
- Responsive
- Forgot Password form
- Profile update form
- Firebase auth and hosting

## Technologies
- **HTML**  
- **CSS** (VANILLA, TAILWIND CSS, DAISYUI)  
- **JAVASCRIPT**  
- **VITE**  
- **REACT**  
- **REACT-DOM**  
- **REACT-ICONS**  
- **REACT-ROUTER**  
- **ANIMATE.CSS**  
- **REACT-HOT-TOAST**  
- **SWIPER**
- **FIREBASE**






## How to Run
1. Download the project
2. Open the folder to your VS code and type npm run dev.
